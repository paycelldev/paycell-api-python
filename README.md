# Requirements

* Python 3
* Django 2.1
* Zeep module to use Soap Clients

# Installation 


```sh
 pip install Django
 pip install zeep

```

# Usage

## Launch Example Application

For only the first time launching the application:
```sh 
 python manage.py makemigration
 python manage.py migrate
```

To run application

```sh 
 python manage.py runserver 0:9090
```
